# Introducción

¡Bienvenido a Nextra! Esta es una plantilla básica de documentos. Puedes usarlo como un punto de inicio para tu propio proyecto :)

## ¿Que es Nextra?

Un framework de generación de sitios **simple**, **poderoso** y **flexible** con todo lo que amas de Next.js.
